## （2）unicode编码：

转换网站：

http://tool.chinaz.com/tools/unicode.aspx  使用 中文 转 Unicode功能，get请求中 将 \ 换成 %

```
union select
uni%u006fn se%u006cect
```



## （3）使用/&#42;!50000&#42;/

/&#42;!&#42;/是内联注释，里面的语句可以被执行

而加入了一串数字，代表的是版本号，版本号高于这个数据才会执行：

```
/*!50001union select*/
```
